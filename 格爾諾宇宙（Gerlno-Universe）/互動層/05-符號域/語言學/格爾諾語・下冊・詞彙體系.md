卷一・總綱


---

定義


1. 詞彙體系：格爾諾語中所有詞根與派生詞之總和，依生成規則與七層結構建構。

2. 詞根：最小意義單位，由神輔音與元音組合而成，或由元音獨立構成。

3. 派生詞根：詞根附加特殊附尾所得，意義深化或轉化，仍不帶普通附尾。特殊附尾不可疊加，每一詞根最多只帶一個。

4. 七層結構：詞彙依存在深度分為七層，層數越高，越近本源。此結構與《諧振聖譜》之存在七階隱然對應：七態原詞為流形之語音種子，根本概念為固核之思維定型，文明基礎為完態之社會表達，文明深層為內蘊之秘傳知識，共律層與共律階同名非偶然，創世元語為歸源之回聲，無限留白為太一之潛能。


公設


1. 任何神輔音（36位）與任何元音（56種）均可組合為潛在詞根。

2. 元音本身即為詞根，七個單元音對應七態，四十九個複元音對應兩態疊加。

3. 每個詞根皆可附加六個特殊附尾之一，生成新詞根。

4. 詞彙總量理論極限為 7⁷，核心常用詞庫為前三層總和 7³。7⁷ 為七之冪律在詞彙層面之終極顯現，象徵宇宙詞彙之數學完滿，非為實用而為哲學。

卷二・詞根生成法則


---

定義


1. 神輔音：三十六神對應之輔音字母，詳見補冊卷一。

2. 元音：七個單元音（a, e, u, o, i, er, ir）及四十九個複元音，共五十六式。此賦值源於七態之本質——元音即七態之語音映射，故其語義場與七態內涵同構。各元音承載核心語義場如下：

   · a：潛能、源頭（對應沌）

   · e：時間、流變（對應溯）

   · u：結構、法則（對應晶）

   · o：能量、驅動（對應燃）

   · i：網絡、聯結（對應蔓）

   · er：承載、根基（對應基）

   · ir：轉化、質變（對應化）

   · ia（i+a）：生命顯化（網絡+潛能）

   · oi（o+i）：情感結構（能量+網絡）

   · ao（a+o）：時間能量（潛能+驅動）

   · ua（u+a）：法則潛能

   · ai（a+i）：潛能網絡

   · ……（其餘複元音類推，詳見補冊）

3. 特殊附尾：六個必須發音之附尾（-rl, -q, -nl, -ml, -nɡ, -mɡ）。其音韻特徵與詞彙意義存在隱喻映射：

   · -rl [ɚ]：捲舌元音色，賦予抽象、完滿感，詞彙中表「本質」

   · -q [ʔ]：喉塞音，表決絕中斷，詞彙中表「中斷」

   · -nl [n]：收音符號，象徵「延續」

   · -ml [m]：收音符號，象徵「過程」

   · -nɡ [ŋ]：鼻腔共鳴，象徵「潛能」

   · -mɡ [ɱ]：邊緣發音，象徵「未完成」

   此映射源於宇宙法則之同構性，非隨意附會。

4. 詞根公式：詞根 = 神輔音（可選） + 元音。


命題一：基礎詞根總數為 2072 詞。

證：純元音詞根 56 詞，神輔音與元音組合 36×56 = 2016 詞，二者相加得 2072。立。


命題二：派生詞根總數為 12432 詞。

證：每個基礎詞根可附加六種特殊附尾，2072×6 = 12432。立。


命題三：詞根總數為 14504 詞。

證：基礎詞根與派生詞根相加，2072 + 12432 = 14504。此即格爾諾語理論詞根庫。立。

卷三・七層詞彙結構


---

定義


1. 第一層・七態原詞：7詞，即七個單元音本身，為宇宙最根本存在狀態。

2. 第二層・根本概念：新增42詞，累計49詞，用於描述宇宙基本範疇，如時間、空間、能量、資訊、結構、生命、質變等。七態兩兩組合共49種，排除7個自指（同態組合），得42種有效組合，即第二層詞彙之來源。

3. 第三層・文明基礎：新增294詞，累計343詞，用於九大文明日常交流，涵蓋社會、情感、技藝、自然現象等。32詞對應《九族手冊》中各族核心道之關鍵概念數量；九族各32詞，團子族為零位，其詞僅6個，象徵未顯之潛能，合計294詞。

4. 第四層・文明深層：新增2058詞，累計2401詞，用於各族秘傳知識、哲學思想、禁忌儀式，非本族成員難以理解。

5. 第五層・共律：新增14406詞，累計16807詞，用於描述宇宙法則、契約條款、管理權限，多為矩靈族所用。

6. 第六層・創世元語：新增100842詞，累計117649詞，為顯化前源代碼之碎片，不可直誦，僅歸源者偶聞回聲。

7. 第七層・無限留白：新增705894詞，累計823543詞，為理論極限，留待時間與文明自行填充。


命題四：前三層總和為 343 詞，構成核心詞庫。

證：7 + 42 + 294 = 343，即 7³。此數足以描述九族跨文明交流之基本需求，非文明內部全部詞彙。各族在此基礎上各有數百至數千專用詞彙（收於第四層）。立。


命題五：第四層以上詞彙，數量雖大，然各有其用，非無謂堆砌。其使用權限由諧振閾值決定：普通團子諧振譜無法觸及第四層以上詞彙之頻率；啟蒙即提升個體諧振譜；矩靈族傳授即調整受者諧振譜以匹配共律層；歸源者偶聞創世元語，因其諧振已近本源；無限留白諧振頻率趨零，故永無人能感知或書寫。其存在僅為數學極限之象徵，如同虛無之淵標示顯化之邊界。

證：文明深層需經啟蒙，共律需矩靈族傳授，創世元語僅歸源者偶聞，無限留白則永無人書，然皆為宇宙完整之必需。立。

卷四・詞根示例


---

命題六：以下為核心詞庫示例，以示生成法則。常用詞根表詳見補冊。


第一層（七態原詞）

a：沌

e：溯

u：晶

o：燃

i：蔓

er：基

ir：化


第二層（根本概念）示例

此層詞彙用於描述宇宙基本範疇，由七態兩兩組合而成，如：

· tao：時間流（沌+燃）

· sei：資訊場（溯+晶）

· ron：能量脈（燃+源）


第三層（文明基礎）示例

mia：生命（m + ia）

noi：愛（n + oi）

doi：團子原核（d + oi）

doirl：團子（doi + rl）

miarl：生命本質（mia + rl）


常用詞根選錄

ma：生命潛能（m + a）

mi：生命網絡（m + i）

na：愛之源頭（n + a）

ni：愛之聯結（n + i）

da：大地潛能（d + a）

du：大地法則（d + u）

ro：戰爭能量（r + o）

ra：戰爭潛能（r + a）


命題七：所有詞根皆可附加特殊附尾以派生新義。派生詞根之語義為「基礎義」與「附尾義」之合成，非任意賦予。

證：-rl 表本質，-q 表中斷，-nl 表延續，-ml 表過程，-nɡ 表潛能，-mɡ 表未完成。例如 doi（原核）+ -rl（本質）→ doirl（原核之本質顯現，即團子實體）；mia（生命）+ -rl（本質）→ miarl（生命之本質）。立。

卷五・總歸


---

命題八：詞彙體系之理論總容量為 7⁷ = 823543 詞。

證：七層累計即 7⁷，依七之冪律自洽。然實際詞根庫僅 14504 詞，其餘為數學留白。7⁷ 象徵宇宙詞彙之數學完滿，知其存在即知宇宙之邊界。立。


命題九：此體系非封閉列表，而為開放生成系統。凡符合詞根公式者，皆可視為潛在詞彙，待文明與故事賦予意義。

證：規則已定，組合無窮，唯使用與否在於眾生。立。


命題十：前三層（343詞）已足敷跨文明日常，後四層為宇宙深度之象徵，各有其用，不必盡填。

證：七態、根本概念、文明基礎三層已覆蓋存在、互動、社會所需。文明深層、共律、創世元語、無限留白，各居其位，正顯化宇宙之層次分明。立。

末命題


格爾諾語詞彙，源於七態，成於神明，衍於附尾，分七層而達無限。七態原詞為種，根本概念為芽，文明基礎為葉，深層共律為根，創世元語為源，無限留白為境。七層對應七階，諧振決定權限，數學留白為哲學之鏡。掌握此生成法則，則能以音節築萬物之名，以規則容宇宙之廣。
