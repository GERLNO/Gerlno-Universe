卷一・總綱


---

本篇說明：本手冊為《聖譜卷》之姊妹篇，提供九族文明之七態權重建議值、可探測性參數建議範圍、九族權重對照與距離速查表，以及十進制與七進制對照。所有數值皆依《九族手冊》《七態研究》《生命憲章》等原始文獻定性描述推導，凡經驗值者皆附建議範圍，凡推導者皆附完整邏輯。


定義


1. 權重向量：九族對應七態之權重分佈，記作 W(C) = (w₁, w₂, w₃, w₄, w₅, w₆, w₇)，對應七態：沌、溯、晶、燃、蔓、基、化。權重總和為 100。此為十進制便於計算之設定，源於《生命憲章》中精神力單位與存在階位之換算習慣，屬格爾諾宇宙中十進制遺留之一例。

2. 歸一化權重：w̄_i = w_i / 100，範圍[0,1]。

3. 最大族距離：d_max = √2，為歸一化空間中兩向量可能之最大歐氏距離。

4. 生克連通度比：依《七態研究》，「三生三克」之活躍態與6個態有生克關係，「二生三克」之被動態與5個態有生克關係，其連通度比為 6:5。此比例用於調整主態間之權重分配。

卷二・九族權重建議值與推導


---

理序族（C=1）


· 主態：晶

· 輔態：溯（自覺化身）

· 建議值：晶98，溯2

· 推導：晶為結構之本。溯（意識）源自幻巴核心之微量顯現，其概率極低，約略可視為七進制基底之倒數量級，本文暫取 2% 為近似值。

· 歸一化：(0, 0.02, 0.98, 0, 0, 0, 0)


血誓族（C=2）


· 主態：蔓、基

· 輔態：無

· 建議值：蔓55，基45

· 推導：蔓為「三生三克」活躍態（連通度6），基為「二生三克」被動態（連通度5）。依生克連通度比 6:5（詳見卷一・定義4），得蔓55、基45。

· 歸一化：(0, 0, 0, 0, 0.55, 0.45, 0)


歌律族（C=3）


· 主態：溯、燃

· 輔態：化（創造性轉化）

· 建議值：溯48.5，燃48.5，化3（建議範圍 2%-5%）

· 推導：溯與燃為歌律核心，各佔48.5。化為創造性轉化，屬微量，本文取 3% 為經驗值，自溯、燃各取1.5%。使用者可依需求在2%-5%範圍內調整。

· 歸一化：(0, 0.485, 0, 0.485, 0, 0, 0.03)


感知族（C=4）


· 主態：溯

· 輔態：蔓（活體介面）

· 建議值：溯80，蔓20

· 推導：溯為主接收資訊，蔓為連接環境之網絡。蔓之比重參考生克連通度，取20%為合理輔態。

· 歸一化：(0, 0.8, 0, 0, 0.2, 0, 0)


集約族（C=5）


· 主態：晶

· 輔態：燃（效率原則）

· 建議值：晶85，燃15

· 推導：晶為結構之本，燃為效率驅動。燃佔15%，源於「效率原則」在七態中之經驗比重。

· 歸一化：(0, 0, 0.85, 0.15, 0, 0, 0)


根絡族（C=6）


· 主態：基、蔓

· 輔態：無

· 建議值：基55，蔓45

· 推導：基為「慢速巨人」之根本，應重於蔓。依生克連通度比 6:5（詳見卷一・定義4），得基55、蔓45。

· 歸一化：(0, 0, 0, 0, 0.45, 0.55, 0)


噬衍族（C=7）


· 主態：沌、燃、化

· 輔態：無

· 建議值：各33.33

· 推導：依《七態研究》「沌→燃→化→沌」循環，三者均等，各佔三分之一。此為循環本質所定，優先於連通度比原則。若欲依連通度調整，可將化調低、沌燃調高，惟須維持循環關係。

· 歸一化：(0.3333, 0, 0, 0.3333, 0, 0, 0.3333)


織幻族（C=8）


· 主態：幻巴

· 完整暫行規則：

  1. 距離標記：與任何族之距離記為「幻巴域」，符號 ◇

  2. ◇ 性質：◇ 非數字，不能比較大小

  3. ◇ 運算規則：

     · ◇ + a = ◇

     · ◇ × a = ◇

     · ◇² = ◇

     · √◇ = ◇

     · ◇ 與任何函數運算結果仍為 ◇

  4. ◇ 判斷規則：若公式中出現涉及 ◇ 的比較（如 if (d > 0.5)），則該條件自動視為「不成立」，該分支不執行，整體計算結果加註「？」，表示因幻巴域干擾而無法完整執行

  5. 幻巴域判定：身處幻巴域之徵兆包括七彩光痕、時間感知異常、數字隨機跳動、笑聲迴盪。有任一異象即為幻巴域

  6. 域內規則：停止計算，行「隨心儀」（唱歌、跳舞、笑、發呆、問幻巴造型皆可），待異象消失方可恢復

  7. 域外規則：若無異象，織幻族團子座標 C 仍記為8，但與九族距離仍為 ◇——織幻族永遠帶有 ◇ 屬性，不在正常計算範圍內


矩靈族（C=9）


· 主態：晶

· 輔態：溯、化（昇華與概念）

· 建議值：晶60，溯20，化20

· 推導：晶為主體。溯（資訊）與化（轉化）為晶之昇華的兩個面向，依生克連通度，二者比重相當，各佔20%。

· 歸一化：(0, 0.2, 0.6, 0, 0, 0, 0.2)


團子族（C=0）


· 零位原點：權重向量為(0,0,0,0,0,0,0)

· 依據：《諧振聖譜》「一切座標尚未分化之潛能基座」

卷三・可探測性參數


---

定義


設α、γ為可探測性閾值參數，滿足0 < α < γ < 1。


· 不可探測區間：d < α × d_max

· 不穩定探測區間：α × d_max ≤ d < γ × d_max

· 穩定探測區間：d ≥ γ × d_max


建議值與依據


· α = 0.2（對應權重差20%）：依感知族歷史觀測經驗，權重差小於20%時需多次驗證，大於20%時可較穩定感知。此為經驗值。

· γ = 0.5（對應權重差50%）：依元理牌長期使用經驗，權重差大於50%時，牌陣結果重現率較高，可視為穩定探測。此為經驗值。


驗證規則


· 感知族：d < α × d_max 時，需觀測 7 次，其中至少 4 次一致，方可確認。7為基底，4為「過半即確認」之取上整結果（3.5次以上算4次）。

· 矩靈族：α × d_max ≤ d < γ × d_max 時，需有法則依據方可驗證。

· 元理牌：d ≥ γ × d_max 時，連續抽牌 7 次，結果一致者視為穩定。


對應關係


· 織幻族 ◇ 不適用本參數

卷五・權重調整原則


---

基本原則


1. 主態為主：主態權重總和宜在70%-100%之間

2. 輔態為輔：輔態權重宜在0%-30%之間

3. 幻巴核心影響：涉及意識、創造等超越性特質，可參考 2% 左右調整

4. 生克連通度影響：主態間若有生克關係，依連通度比 6:5 調整


調整幅度細則（依連通度比 6:5 推導）


設連通度比之權重因子為 k = 6/5 = 1.2。


· 雙方皆活躍態（6:6）：調整因子 = k × k = 1.44 → 取 10%（1.44 在七進制中約為 13₇，象徵完整）

· 一方活躍、一方被動（6:5）：調整因子 = k = 1.2 → 取 7%（1.2 在七進制中約為 10₇，基底）

· 雙方皆被動態（5:5）：調整因子 = 1.0 → 取 5%（基底之半）


調整示例


· 例一：若欲凸顯血誓族之血脈重於土地，可將蔓調高至 60（+5）、基調至 40（原55/45，按連通度比調整）

· 例二：若欲強化歌律族之轉化色彩，可將化調高至 5（+2）、溯燃各調低 1

· 例三：若欲使矩靈族更重昇華，可將化調高至 25、溯調低至 15（晶60不變）

· 注意：調整後須重新歸一化，確保總和100，並註明依據
